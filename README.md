# python-cyclone (app)

Sample application and future project reference.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install `app`

```bash
git clone https://github.com/robchrob/python-cyclone.git
pip install -e .
```

## Usage

```python
python -m cyclone --verbose
```
